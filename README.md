# portal-wap-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，这些页面主要面向移动端（WAP）环境下的信息展示与交互场景。仓库本身不绑定任何特定域名或网站，仅作为一组静态页面的集中存储与版本管理。

## 目录结构

```
portal-wap-guide/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── docs/           # 相关文档或说明文件
└── README.md       # 本文件
```

- `pages/`：每个子文件夹或文件代表一个独立页面，页面之间无依赖关系。
- `assets/`：按需存放页面所需的公共或私有资源。
- `docs/`：可选的补充文档，用于记录页面设计思路或使用注意事项。

## 页面归档说明

- 所有页面均为纯前端 HTML 文件，不依赖后端服务。
- 页面内容与样式独立，可直接在浏览器中打开使用。
- 归档目的为方便查阅、分发或离线使用，不包含任何形式的诱导点击或商业推广。
